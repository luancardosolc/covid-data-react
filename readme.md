# Covid Data

React app to visualize covid data from [covid-19-data](https://github.com/owid/covid-19-data/blob/master/public/data/latest/owid-covid-latest.json).

## Run without docker

```bash
# install dependencies
yarn

# run the app
yarn dev
```

#### Open the URL bellow in your browser
http://localhost:3000/

## Run with docker

```bash
docker compose up

```
#### Open the URL bellow in your browser
http://localhost:3000/
